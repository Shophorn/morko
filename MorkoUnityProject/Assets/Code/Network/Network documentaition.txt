Message construction

Game Update		MORKO 	cmd (0 - 49)		no key	args	data?
Game Event		MORKO 	cmd (50 - 99)		key		args	data?
App Event		MORKO 	cmd (100 - 149)		key		args	data?
Menu Event		MORKO 	cmd (150 - 199)		key		args	data?


Parsing:
	1. Check if stars with MORKO
	2. Get command
		2.5 if command is in event range, check key
	3. Convert arguments to proper structure
	4. Profit

Key:
	8 bits from playerID so that we get unique key for each player
	24 bits from ever incrementing counter
